# Projet
projet Montréal Mic'd up

Un lien vers la zone d'administration se trouve en bas de la page d'accueil. Cliquer dessus mene a la page d'inscription. Remplir le formulaire avec un courriel, un mot de passe, un prenom et un nom, puis soumettre. Vous serez ensuite redirige automatiquement vers la page de connexion. Entrer le courriel et le mot de passe choisis lors de l'inscription pour acceder a l'administration.

Une fois connecté, il est possible d'ajouter, modifier ou supprimer des artistes.
L'application est disponible en francais et en anglais. Les boutons FR et EN dans la barre de navigation permettent de changer la langue.
